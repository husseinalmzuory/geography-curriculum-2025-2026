# دليل تحديث مناهج أقسام الجغرافية 2025-2026

موقع تفاعلي عربي مناسب للموبايل لعرض مناهج أقسام الجغرافية في كليات التربية للعلوم الإنسانية.

## المحتويات

- `index.html`: الصفحة الرئيسية.
- `styles.css`: تنسيق الموقع وتصميم الموبايل.
- `app.js`: منطق التفاعل والبحث وعرض المواد.
- `data-pages.json`: نصوص صفحات الدليل المستخرجة للبحث والعرض.
- `assets/geography-curriculum-2025-2026.pdf`: ملف PDF الأصلي.
- `.nojekyll`: لتعطيل معالجة Jekyll في GitHub Pages.
- `manifest.webmanifest` و `sw.js`: دعم أولي لتجربة تطبيق ويب.

## النشر على GitHub Pages

1. أنشئ مستودعاً جديداً في GitHub باسم مثل:
   `geography-curriculum-2025-2026`
2. ارفع كل ملفات هذا المجلد إلى المستودع، وليس المجلد الأب.
3. ادخل إلى:
   Settings > Pages
4. اختر:
   Source: Deploy from a branch
5. اختر:
   Branch: main
   Folder: /root
6. اضغط Save.
7. بعد دقيقة أو دقيقتين سيظهر رابط الموقع.

## ملاحظة تدقيق

تم اعتبار المحتوى مدققاً من صاحب الملف قبل التجهيز للنشر. توجد مواد مذكورة في الجداول لكن لم تظهر لها مفردات تفصيلية مستقلة في ملف PDF، لذلك تظهر داخل الموقع مع تنبيه مناسب.
